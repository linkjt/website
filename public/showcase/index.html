JSDB Showcase

Hello, I know this website looks like poo, but really this is meant to be a showcase of JSDB, a new light wieght markup language for Javascript Based Forums. You see, many existing forums use a markup language called bbcode, but it is kinda old and mostly for forums that interact with php jsdb changes this, and works as a easily useable markup up language for any application, mesaging apps, forums, or even a contact form. It works similar to bbcode, and to use it like this [tag]text[/tag]. There are a few tags as of now, bold, italic, image, emoji's, ect. The Emojis work the same, you just say [e](insert feeleing)[e], right now there is funny, mad, ok, and sad. All of the buttons bellow are included in jsdb's functionality, so try it out. If you want to add it to your project, download it bellow, then Move it to your project, and then use the rules.txt to learn how to map the files
